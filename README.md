# eigo
コマンドラインで動くちょびっと辞書

## Usage
英和としてつかう
```
eigo interpolation

eigo interp            # ..orlation? ..oration? ってなったとき用

eigo ^interp           # 先頭マッチ

eigo ^interpolation$   # 完全マッチ
```

和英としてつかう
```
eigo 論文

eigo ^論文           # 日本語でも大丈夫
```

## Getting Started
```
$ git clone https://github.com/kottn/eigo
$ cd eigo; ./setup
```
